# Python AI Assistant 🤖

A simple Python-based AI Assistant with automation features like Google search, YouTube control, WhatsApp messaging, and voice responses.

## Features
- Voice Responses
- Open Google
- Open YouTube
- Play YouTube Songs
- Google Search
- Open Notepad
- Open Calculator
- Send WhatsApp Messages

## Technologies Used
- Python
- pywhatkit
- pyttsx3

## Installation

```bash
pip install pywhatkit pyttsx3
```

## Run

```bash
python main.py
```
